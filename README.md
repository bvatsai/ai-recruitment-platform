
# Enterprise AI Recruitment Platform

## Overview
A production-oriented AI recruitment platform being built step-by-step using Python, FastAPI, PostgreSQL, and LLMs, that helps:

- Job seekers improve resumes
- Recruiters find the best candidates
- AI match resumes with job descriptions
- RAG-powered recruitment assistant
